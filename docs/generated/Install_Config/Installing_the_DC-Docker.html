Run Data Collector from Docker

You can run the Data Collector image from Docker Hub.

To run the Data Collector image from Docker Hub, use the following command in the Docker Quickstart Terminal:
docker run -d -P --name streamsets-dc streamsets/datacollector

For more information about running Data Collector from Docker, see https://hub.docker.com/r/streamsets/datacollector/.
