HDFS Requirements

Use the following requirements to configure a cluster mode pipeline to read from HDFS:
  1. Install HDFS.
  2. Install the Data Collector on the YARN gateway node.
  3. In the pipeline properties, on the General tab, set the Execution Mode property to Cluster.
  4. In the pipeline, use the Hadoop FS origin.
