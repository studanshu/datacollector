Importing Pipelines

Import pipelines to use pipelines developed on a different Data Collector or to restore backup files. Import pipelines from pipeline files. Pipeline files are JSON files exported from a Data Collector instance.
  1. From the Home page or Getting Started page, click Import Pipeline.
  2. In the Import Pipeline dialog box, enter a pipeline name and optional description. Browse and select the pipeline file and click Import.
